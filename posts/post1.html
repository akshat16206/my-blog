The website: mY-bLoG

Posted on June 24, 2025

This website is a small personal project I built to organize and share my thoughts, technical projects, research paper summaries, and everything else I’m learning.

The visual style i created is inspired by the old tech boom of the 1990s — when information was simple, direct, and text-focused. I wanted something minimal and distraction-free, where content matters more than design effects.

The purpose of this site is simple:

  • 🧠 Share ideas and projects I’m working on
  • 📚 Document papers and topics I’m learning
  • 📝 Reflect on thoughts and insights publicly

The entire website is built using just HTML, CSS, and JavaScript — no frameworks, no clutter. If you’d like to see how it works or use it for your own personal space, the source code is available here:
